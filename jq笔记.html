jq事件模型

添加事件处理
					on(eventType[,selecetor][,data],handler)
						事件名称  选择器      数据	事件处理函数	
					事件委托：可以把事件绑定在父元素上面 		
					
					例如给ul添加函数
						$('ul').on('click',function(event){
						console.log('%O clicked',$(event.target));
					});
					备注：%o为转义字符，会用逗号后元素替换掉

					阻止冒泡： stopPropagation()
					阻止默认行为： preventDefault()
					阻止冒泡和默认行为： return false
